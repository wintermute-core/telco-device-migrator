# Migration tool

Services migration tool

## Tech stack

  * Java 11
  * Spring Boot
  * Kie
  * Drools
  * JUnit
  * Lombok
  * Swagger

## Development

Local execution:
```
./gradlew bootRun
```

Swagger UI:
http://localhost:8080/swagger-ui/index.html

Tests execution
```
./gradlew clean test
```
